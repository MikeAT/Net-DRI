Net-DRI-0.96_06_tdw

  • based on Net-DRI-0.96_05 (see README)

This version contains mostly updates related new gTLDs:

  • LaunchPhase support
  • Support for multiple new gTLD backend providers (with more being added, and yes some bits being fixed)
    • Afilias
    • ARI
    • CentralNIC
    • CoreNIC
    • Charleston Road Registry
    • Donuts
    • Famous Four Media
    • GMO Registry
    • KSRegistry / incl. Starting Dot
    • Minds And Machines
    • Neustar
    • Registry-in-a-Box from NicAT
    • Rightside
    • Tango from KNIPP
    • UniRegistry
  • Limited support for new gTLD services
    • TMCH (Deloitte): Curretly V1 only
    • TMDB (IBM): CNIS and SMDRL only
  • Added/Updated/Fixed support for ccTLDs
    • UK
    • IT
    • ES
    • NO
    • SE/NU (IIS module)
    • BE
    • EI
    • DE
    • CH
    • ZA
  • Includes a few other bug fixes
